# cost.calc — Калькулятор себестоимости мебели

Локальное веб-приложение для расчёта себестоимости продаж на основе выгрузок из **1С:УНФ**.  
Работает полностью в браузере — файлы никуда не отправляются.

---

## Что делает

Вы загружаете три файла из 1С и получаете по каждой проданной позиции:

- себестоимость на единицу и итого
- маржу в рублях и процентах
- статус расчёта (полный / неполный / нет спецификации)

**Ключевая особенность** — рекурсивный расчёт через полуфабрикаты. Если в составе стула стоит «Каркас Гольф», приложение автоматически раскрывает его спецификацию до уровня чистого сырья (до 8 уровней вложенности).

---

## Быстрый старт

1. Скачайте файл `sebestoimost.html`
2. Откройте в браузере (Chrome, Edge, Firefox, Safari)
3. Загрузите три файла из 1С
4. Нажмите **▶ Рассчитать**

---

## Входные файлы

### 1. Продажи
Путь в 1С: `Продажи → Реализация товаров и услуг → сохранить xlsx`

| Колонка | Пример |
|---|---|
| Номенклатура | Стул Эдвин жестк |
| Характеристика | Белый, эмаль |
| Количество | 12 |
| Выручка / Сумма | 84 000 |

> Приложение находит колонки автоматически по названию — точное расположение не важно.

---

### 2. Цены закупок
Путь в 1С: `Закупки → Анализ закупок → сохранить xlsx`

| Колонка | Пример |
|---|---|
| Номенклатура | Фанера 15 |
| Цена / Стоимость | 1 200 |

---

### 3. Спецификации — **единый файл со всеми уровнями**
Путь в 1С: `Все функции → Универсальный отчёт`

Настройки отчёта:
- **Тип объекта:** Регистр сведений
- **Имя объекта:** Спецификации по умолчанию
- **Структура:** группировка по `Номенклатура + Характеристика + Спецификация`
- **Поля:** `Спецификация.Состав.Номенклатура`, `Состав.Ед.`, `Состав.Количество`

Выгрузите **все спецификации сразу** — один файл содержит и готовые изделия, и полуфабрикаты.

---

### Дополнительно: Остатки со себестоимостью *(опционально)*
Нужен если сырьё есть на складе, но давно не закупалось и цены нет в анализе закупок.

Путь в 1С: `Склад → Остатки и доступность товаров → колонка "Себестоимость" → xlsx`

## Как работает расчёт

```
Продажи (что продали × сколько)
    ↓
Спецификация изделия
    ├── Материал А × 0.44 кг → цена из закупок → сумма
    ├── Полуфабрикат Б × 1 шт
    │   ├── Материал В × 2.4 л → цена из закупок → сумма
    │   └── Материал Г × 0.01 кг → цена вручную → сумма
    └── Материал Д × 0.5 м → цена из остатков → сумма
         ↓
Себестоимость/ед = сумма всех компонентов
Себестоимость продаж = себестоимость/ед × количество
Маржа = Выручка − Себестоимость продаж
```

**Приоритет источников цены:** ручная цена > цена из остатков > цена из закупок

## Вкладки приложения

| Вкладка | Описание |
|---|---|
| **Данные** | Загрузка трёх файлов из 1С |
| **Цены сырья** | Таблица всего сырья — видно где нет цены, можно ввести вручную или загрузить остатки |
| **Расчёт** | Основная таблица: выручка / себестоимость / маржа по каждой позиции |
| **Детализация** | Дерево себестоимости по выбранной модели — каждый компонент с долей в % |
| **Отчёт** | Сводка: топ убыточных, топ прибыльных, список проблем |

---

## Цветовая индикация в таблице расчёта

| Цвет | Значение |
|---|---|
| 🟢 Зелёная полоса | Расчёт полный, маржа положительная |
| 🔴 Красная полоса | Маржа отрицательная (убыток) |
| 🟡 Жёлтая полоса | Нет цены на часть сырья — себестоимость занижена |
| ⚫ Серая полоса | Спецификация не найдена |

## Экспорт

Кнопка **↓ Excel** формирует файл `sebestoimost_ДД-ММ-ГГГГ.xlsx` с тремя листами:

1. **Расчёт себестоимости** — полная таблица результатов
2. **Нет цены на сырьё** — проблемные материалы и в каких изделиях встречаются
3. **Цены сырья** — все цены с разбивкой по источнику

---

## Типичные ситуации

**Статус «нет спецификации»**  
Проверьте точное написание номенклатуры — оно должно совпадать в файле продаж и в спецификациях.

**Статус «неполный расчёт»**  
На часть сырья нет цены. Перейдите в «Цены сырья» → фильтр «Нет цены» → введите вручную или загрузите остатки. Себестоимость занижена ровно на стоимость этих материалов.

**Маржа отрицательная**  
Проверьте актуальность цен в файле закупок и нормы расхода в спецификации.

**Сырьё давно не закупалось**  
Загрузите файл остатков или введите цену вручную на вкладке «Цены сырья».

## Технически

- Чистый HTML + JavaScript, без серверной части
- Парсинг xlsx через [SheetJS](https://sheetjs.com/)
- Работает офлайн после первой загрузки (кроме шрифтов)
- Протестировано на файлах 1С:УНФ: 1700+ спецификаций, 12 000+ компонентов

---

> Приложение считает **нормативную себестоимость** по спецификациям и ценам закупок.  
> Фактические отклонения (брак, перерасход) не учитываются.
